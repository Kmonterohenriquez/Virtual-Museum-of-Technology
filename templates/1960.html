1964 – First computer mouse – Douglas Engelbort
Computer Mouse:

The first computer mouse was invented by Douglas Engelbart in 1964. While we’re all probably used to using a wireless mouse or a trackpad on a laptop, the first mouse had a wooden shell, circuit board, and two metal wheels that connected with the surface beneath it to control direction. This mouse was first publicly demonstrated at the 1968 Fall Joint Computer Conference. The first computer mouse led the way for further innovation including the “Ball Mouse”, optical mouse, and trackpads.

1968 – DRAM (RAM) – Robert Dennard
DRAM (RAM):

Dynamic Random Access Memory (DRAM), or RAM, is a type of computer memory that temporarily stores data. So, it’s basically the computer’s working memory. RAM is essential to the overall functionality of a computer because the more RAM a computer has, the more information the computer is able to work with at the same time. RAM was first patented by Robert Dennard in 1968. The Intel 1103 memory chip, which first came out in 1970, was the first DRAM chip released and by 1972 was the best-selling semiconductor memory chip in the world.
